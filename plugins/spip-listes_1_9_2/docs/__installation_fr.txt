/**********************************************************/
/*                    SPIP-Listes                         */
/**********************************************************/

   - Permet de fabriquer des newsletters, manuellement ou automatiquement, à partir d'un site sous Spip (http://spip.net/)

Pour ce faire SPIP-Listes vous permet de :
   * rédiger des messages,
   * gérer des listes d'abonnés, chaque abonné étant libre de s'abonner ou se désabonner seul et de choisir son format de message (html ou texte),
   * envoyer des informations par  e-mail, à fréquence régulière si vous le désirez,
   * envoyer automatiquement les dernières parutions du site, ou un squelette spip personnalisé
   * modifier à votre convenance la mise en page de vos envois.

   - La todo list se trouve sur le site bloog.net : http://bloog.net/spip.php?article115


/**********************************************************/
/*              Installation de SPIP-Listes               */
/**********************************************************/

I. Installation des fichiers du plugin SPIP-Listes

1) Décompressez l'archive .zip

2) Copiez par FTP le repertoire obtenu spip-listes/ (et ses sous-répertoires), sans changement, en conservant l'arborescence, dans le répertoire /plugins (que vous devez créer s'il n'existe pas).


II. Appel de SPIP-Listes dans vos squelettes


1)  Ajouter le code #FORMULAIRE_ABONNEMENT dans le squelette ou vous souhaitez afficher le formulaire d'inscription.
Vous pouvez utiliser un squelette personnel à la place de formulaire_abonnement.html (pour gérer l'inscription à une liste particulière par exemple) en utilisant la notation #FORMULAIRE_ABONNEMENT{mon_squelette}.

Note : La page abonnement.html gère des modifications/désinscriptions des abonnés.

2) Dans le repertoire du plugin, éditer les fichiers /patrons/nouveautes.html et /patrons/patron_simple.html, et adapter les adresses des images à votre site.
Attention, en cas de sélection fondées sur des boucles sur l'âge, le squelette doit retourner un message vide en cas d'erreur.
Vous pouvez créer tous les patrons que vous désirez, en les déposant dans ce répertoire, ils apparaîtront automatiquement dans les choix de la partie privée.

III. Utilisation de SPIP-Listes, dans la partie privée

1) Cliquer, dans le menu de configuration, sur Gestion des plugins.

2) Sur la ligne Plugin SPIP-Listes cocher la case Activer le plugin, puis valider

3) Cliquer, dans le menu Edition, sur lettres d'information (ou accédez directement à la page par http://votre_site.net/ecrire/?exec=spip_listes)

Vous êtes prêt :

4) Créer une(des) liste(s) de diffusion, puis des abonnés (avec possibilité d'import d'une liste d'abonnés).
