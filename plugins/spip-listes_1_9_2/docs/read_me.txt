If you want to use the template version of abonnement.php3, which in turn uses
the new form tag #FORMULAIRE_MODIF_LISTES, you have to overwrite the old
abonnement.php3 file with abonnement.php3.txt that you can find in this folder.
Just don't forget to rename it abonnement.php3.

Renato Formato


Si vous voulez utiliser la version squelette de abonnement.php3, 
qui utilise maintenant une balise de formulaire #FORMULAIRE_MODIF_LISTES, 
vous devez écraser l'ancien fichier abonnement.php3 
avec le fichier abonnement.php3.txt que vous trouverez dans ce répertoire.
N'oubliez pas de le renommer en abonnement.php3
